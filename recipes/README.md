### 🍳 Recipes

#### 🧭 Project Overview

This project is part of **The Odin Project’s Full-Stack Developer curriculum**.  
It is a **basic recipe website** built entirely with **HTML**, designed to practice fundamental web development concepts before introducing CSS and JavaScript.

The website includes:
- A **main index page** with links to multiple recipe pages.  
- Each **recipe page** presents ingredients, preparation steps, and navigation links.

> ⚠️ The site’s layout is intentionally minimal — the main goal is to strengthen **HTML structure, semantic markup, and linking**.  
> CSS styling will be added later in the curriculum.

---

#### 🛠️ Skills Demonstrated

By completing this project, I practiced:
- Writing clean, semantic HTML  
- Creating and managing internal/external hyperlinks  
- Structuring content with headings, paragraphs, and lists  
- Organizing project files logically across multiple pages  
- Using **Git** for version control and **GitHub** for project hosting  
- Following a professional commit workflow with meaningful messages
