## 🧰 Technologies & Concepts

- **HTML5** — semantic markup  
- **CSS3** — layout, responsive design, Flexbox/Grid  
- **JavaScript (ES6+)** — DOM manipulation, events, objects  
- **Vite** — modern frontend tooling for development and builds  
- **Git & GitHub** — version control and project organization  

---

## 📘 About The Odin Project

**The Odin Project** is a free, open-source curriculum focused on teaching web development through hands-on, real-world projects.  
Its project-driven approach emphasizes problem-solving, code readability, and incremental skill development.
