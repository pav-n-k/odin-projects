### ✂️ Rock Paper Scissors

#### 📚 Introduction

This project is part of The Odin Project - Foundations Course and focuses on building a classic Rock Paper Scissors game. It serves as an introduction to JavaScript fundamentals and problem-solving approach.

#### 🎮 Project Overview

- Console-based game against the computer
- 5-round matches with score tracking
- Pure JavaScript implementation
- Focus on core programming concepts: functions, variables, game logic

#### 🛠️ Technologies Used

- HTML5 (minimal setup)
- JavaScript (ES6+)
- Git & GitHub (version control and project management)

#### 🚀 Features

- Random computer choice generation
- Case-insensitive user input handling
- Round-by-round winner determination
- Final score calculation and game winner announcement

#### 📁 Project Structure

The project consists of a single HTML file linked to an external JavaScript file containing all game logic, following clean code separation practices.
